### jssdk rxjs 封装 Iwe7JssdkService
| 方法             | 说明                                   |
|----------------|--------------------------------------|
| load           | 加载cdn上得jssdk并初始化                     |
| loadObservable | 加载cdn上得jssd,远程获取jssdk初始化配置，并初始化jssdk |


### Iwe7JssdkRecordService
| 方法     | 说明                         |
|--------|----------------------------|
| record | 开始录音，监听录音完成或停止录音，返回localId |
| stop   | 停止录音                       |
